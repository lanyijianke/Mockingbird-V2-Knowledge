# 情报站 + 品牌重定位 设计规格

> 日期：2026-05-04
> 状态：待审核

## 1. 背景与目标

知更鸟知识库现有内容丰富（AI/金融资讯、提示词库、热榜、学社深度内容），但品牌定位偏窄（"情报团队"），且学社中的情报内容（快讯、叙事追踪）锁在会员区，对公开流量没有吸引力。

**目标：**
1. 调整品牌定位，从"情报团队"升级为"AI 驱动的综合信息平台"
2. 新增公开情报站页面，用摘要+额度限制的模式引流和转化
3. 首页增加情报站入口，形成三条主线：AI / 金融 / 情报站

## 2. 品牌重定位

### 标语变更

| 项 | 当前 | 新 |
|---|---|---|
| slogan | 一群 AI 智能体组成的情报团队 | AI 智能体团队 7×24 为你追踪有价值的信号 |
| 描述 | 帮你从信息洪流中，看见真正重要的东西 | 不变 |

### 首页变更

- Hero 区标语替换为新文案
- 入口按钮从 2 个变为 3 个：AI / 金融 / 情报站，等宽并排
- 团队区、logo、数字雨背景保持不变

## 3. 情报站页面

### 3.1 路由

`/intel` — 品牌门户下一级独立页面

### 3.2 页面布局：卡片网格

```
┌─────────────────────────────────────────┐
│  筛选标签：[全部] [快讯] [叙事]  · 时间范围 │
├─────────────┬───────────────────────────┤
│ ┌─────────┐ │ ┌─────────────────────┐   │
│ │ 叙事卡片 │ │ │    快讯卡片          │   │
│ │ 品牌色边框│ │ │    简洁边框          │   │
│ └─────────┘ │ └─────────────────────┘   │
│ ┌─────────┐ │ ┌─────────────────────┐   │
│ │ 快讯卡片 │ │ │    叙事卡片          │   │
│ └─────────┘ │ └─────────────────────┘   │
└─────────────┴───────────────────────────┘
```

- 顶部筛选标签栏：全部 / 快讯 / 叙事，右侧可选时间范围
- 卡片网格，2-3 列响应式布局
- 内容按时间倒序混排（叙事和快讯交错展示）

### 3.3 叙事卡片

- 品牌色（青色）边框突出显示
- 显示内容：
  - 生命周期阶段标识：萌芽（灰）→ 上升（青）→ 高峰（白）→ 冷却（暗）
  - 热度分数（0-100）
  - 叙事标题（一句话）
  - 关联情报数量
  - 核心实体标签（如有）
- 点击展开完整叙事详情（受额度限制）

### 3.4 快讯卡片

- 简洁边框，低调
- 显示内容：
  - 时间戳
  - 一句话摘要
  - 来源标签
- 点击展开完整内容（受额度限制）

### 3.5 数据源

复用现有 API 接口：
- 快讯：`/api/academy/quicknews`
- 叙事：`/api/academy/narratives`

接口已有完整数据：AI 摘要、质量评分、热度、生命周期阶段、事件链等。情报站页面只做展示层，不修改后端逻辑。

## 4. 访问控制

### 4.1 分层模型

| 用户状态 | 能看到的 | 额度 |
|---------|---------|------|
| 未登录 | 卡片网格完整可见（标题、时间、标签），点击卡片详情时弹出登录引导 | 无 |
| 普通注册用户 | 完整列表 + 卡片详情（AI 摘要、事件链等） | 每日 N 条（具体数值待定） |
| 付费会员 | 完整列表 + 卡片详情 + 学社全部内容 | 无限制 |

### 4.2 实现方式

- 页面本身不需要登录即可访问（middleware 不拦截 `/intel`）
- API 层控制：未登录请求只返回摘要字段（标题、时间、标签），详情字段为空
- 点击卡片时前端检查登录状态，未登录则弹出登录引导（非跳转，用模态框）
- 普通用户额度计数存储在数据库，每日重置
- 达到额度上限时展示升级提示

## 5. 不在本次范围内

- 学社内部页面改造

- SEO 元数据优化（后续可独立处理）
- 移动端专项适配（响应式覆盖即可）
