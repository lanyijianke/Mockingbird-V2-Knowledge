# 情报站 Tab 切换布局设计

## 背景

情报站当前拆分为两个独立页面（`/intel/news` 和 `/intel/narratives`），用户反馈希望合并为一个页面，同时要体现 Agent 正在工作、支持未来扩展（KOL 追踪等）。

## 设计决策

1. **Tab 切换布局** — 单页面内 Tab 栏切换内容面板，新增内容类型只需加 Tab
2. **Agent 卡片头部** — 展示四个智能体的实时工作状态
3. **统计汇总条** — 今日情报总数、已处理、待处理、排队中
4. **路由合并** — `/intel` 为唯一入口，Tab 切换为客户端状态不刷新页面

## Agent 卡片区

四个 Agent 各一张卡片，包含：
- **头像图标** — emoji + 渐变背景圆圈 + 彩色边框
- **名称 + 状态** — 绿色"● 爬取中"/"● 分析中"或灰色"○ 空闲"
- **计数** — 已处理 N / 待处理 M
- **进度条** — 渐变色，宽度 = 已处理/(已处理+待处理)

| Agent | 图标 | 职责 | 边框色 |
|-------|------|------|--------|
| 哨兵 | 📡 | AI 前沿侦察 | #00d4ff cyan |
| 金脉 | 💹 | 金融情报分析 | #ff9500 orange |
| 织言 | ✍️ | 提示词工程 | #a855f7 purple |
| 探针 | 🔍 | 趋势数据监测 | #00ff88 green |

**数据来源** — 新增 API `/api/academy/agent-status`，返回每个 agent 的 processed/pending 计数和当前状态。初始阶段可硬编码或从 system_logs 表聚合。

## 统计汇总条

Agent 卡片下方，单行展示：
- 今日情报总数
- 已处理数（绿色）
- 待处理数（橙色）
- 排队中数（灰色）

数据同来自 agent-status API。

## Tab 栏

- **快讯** — 当前已有，带数量角标
- **叙事** — 当前已有，带数量角标
- **KOL追踪** — 灰色，标注"即将上线"
- 未来新增类型直接追加 Tab

默认激活"快讯" Tab。移动端 Tab 横向滚动。

## Tab 内容面板

### 快讯 Tab

沿用现有 `intel-news` 页面的设计：
- 按时间/质量排序切换
- 按日期分组（今天/昨天/日期）
- 每条：时间、来源、质量评分(★)、NEW 标记、标题、摘要(2行)、标签
- 点击展开详情 + 原文链接
- 付费墙：免费 20 条，会员全量

### 叙事 Tab

沿用现有 `intel-narratives` 页面的设计：
- 阶段统计条（巅峰/升温/初现/冷却）
- 分类筛选（全部/AI/Web3/金融）
- 双列卡片网格
- 每张卡片：排名、分类徽章、阶段、信号强度、热度进度条、摘要、实体标签、文章数
- 付费墙：免费 5 条，会员全量

### KOL Tab（占位）

灰色占位区域，"正在构建中，敬请期待"。

## 路由变更

| 旧路由 | 新路由 | 说明 |
|--------|--------|------|
| `/intel` → redirect `/intel/news` | `/intel` | 主页面，含所有 Tab |
| `/intel/news` | 删除 | 合并进 Tab |
| `/intel/narratives` | 删除 | 合并进 Tab |
| 导航链接"叙事""快讯" | 导航链接"情报站" → `/intel` | 单一入口 |
