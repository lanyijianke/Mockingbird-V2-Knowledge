设计要点总结

页面结构

  • 顶部导航（全局复用）
  • Agent 卡片区（4 张卡片）
  • 统计汇总条
  • Tab 栏（快讯 | 叙事 | KOL...）
  • Tab 内容面板

扩展方式

  • 新增内容类型 → 加一个 Tab
  • 新增 Agent → 加一张卡片
  • 统计维度 → 改汇总条
  • 每个 Tab 内部布局独立
  • 移动端 Tab 横向滚动

付费墙

  • 快讯：免费 20 条，会员全量
  • 叙事：免费 5 条，会员全量
  • 未来 Tab 同理按类型设限额
  • Agent 头部始终可见（免费也要看到）

路由结构

  • /intel → 情报站主页（含 Tab）
  • Tab 切换不刷新页面（客户端状态）
  • 导航栏只有一个"情报站"链接
  • Agent 状态数据来自 API
